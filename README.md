# Expense Tracker

A clean, modern, mobile-first personal finance tracker with real-time balance calculations, category-based expense tracking, and income/expense visualization.

## Features
- Add, edit, and delete income/expense transactions
- Real-time balance and category breakdown
- Responsive, mobile-first design
- Built with vanilla HTML, CSS, and JavaScript (no frameworks/dependencies)

## Run Locally
No build step or dependencies required.

1. Clone or download this repository
2. Open `index.html` directly in your browser, **or**
3. Use a local server (recommended) — e.g. VS Code's Live Server extension

## Tech Stack
- HTML5
- CSS3 (custom properties/variables)
- JavaScript (ES6+)
