# CoinCraft — Expense Tracker

A clean, modern personal finance tracker for managing income and expenses, with real-time balance tracking and category-wise breakdowns.

## Features
- Add and delete income/expense transactions
- Real-time balance calculation
- Category-based spending breakdown (Food, Bills, Transport, Shopping, and more)
- Searchable and filterable transaction history
- Multi-currency display support
- Fully responsive design

## Tech Stack
- HTML5
- CSS3
- JavaScript 

## Run Locally
No installation or build step required.

1. Clone or download this repository
2. Open `index.html` directly in your browser
